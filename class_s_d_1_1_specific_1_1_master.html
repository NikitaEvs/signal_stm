void SD::Specific::Master::EnableClocking ( ) const
inlineoverridevirtual

System clock configuration The system clock is configured as follow:

  • System clock source = PLL (HSE / 1)
  • SYSCLK(MHz) = 72
  • HCLK(MHz) = 72
  • AHB prescaler = 1
  • APB1 prescaler = 2
  • APB2 prescaler = 1
  • HSE frequency(MHz) = 8
  • PLLMUL = 9
  • Flash latency = 2
